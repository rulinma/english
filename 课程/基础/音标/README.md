# 英语国际音标

## 音标

国际音标（48）

* 元音（20）
  * 单元音（12）
    * 长元音
      * [i:]、[ɔ:]、[u:]、[ə:]、[ɑ:]
    * 短元音
      * [i]、[ɔ]、[u]、[ə]、[ʌ]、[e]、[æ]
  * 双元音（8）
    * [ei]、[ai]、[ɔi]、[ɛə]、[uə]、[iə]、[au]、[əu]
* 辅音（28）
  * 清辅音（10）
    * [p]、[t]、[k]、[f]、[s]、[ʃ]、[θ]、[tʃ]、[tr]、[ts]
  * 浊辅音（10）
    * [b]、[d]、[g]、[v]、[z]、[ʒ]、[ð]、[dʒ]、[dr]、[dz]
  * 鼻音（3）
    * [m]、[n]、[ŋ]
  * 似拼音（3）
    * [h]、[l]、[r]
  * 半元音（2）
    * [j]、[w]
