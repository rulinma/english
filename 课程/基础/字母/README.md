# 字母

## 前言

* 首先有大写字母，后才有小写字母。
* 大写字母主要来自于象形及其衍生意义（人类认识事物的一般特征）。
* 小写字母主要是为了书写方便。
* 英文字母起源拉丁文字，最初有21个字母，引入了希腊字母Y和Z，从I分离出J，从U分化出V，两个V变成W。
