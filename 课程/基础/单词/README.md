# 英语单词

## 简介

单词由字母组成，自然有其字母意义的衍生和表达。同时单词也有些共同的词根，前缀和后缀，类似于汉字的偏旁部首，能够熟练的掌握字母象形及其衍生意义，同时结合起来有助于快速学习、记忆和拓展词汇量，而且能事倍功半，
