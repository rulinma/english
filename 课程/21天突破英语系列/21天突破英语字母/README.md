# 21天突破英语字母

## 前言

* 首先有大写字母，后才有小写字母。
* 大写字母主要来自于象形及其衍生意义（人类认识事物的一般特征）。
* 小写字母主要是为了书写方便。
* 英文字母起源拉丁文字，最初有21个字母，引入了希腊字母Y和Z，从I分离出J，从U分化出V，两个V变成W。

  * 与字母 C 表达的洞穴不同，字母 O 表达的多是与“口”有关的意思。例如，口述的(oral)、歌剧(opera)、服从(obey)等。从形状上看，字母 O 与字母 C 有些相似，但又有点区别。从两组单词中也许可以找到 C 和 O 的区别，例如，小圈(circle)和轨迹(orbit)、(卷心菜) cabbage 和洋葱(onion)。含有字母 C 的单词 circle，无论是 用来表达马戏团的“火圈”还是《西游记》中孙悟空给唐僧画的“保护圈”，像字母 C 一样似乎有个“出 口”;含有字母 O 的 orbit，则通常用来表示星球的运行轨迹，像字母 O 一样没有“出口”。另外，cabbage 和 onion 的叶片，一个是有“开口”的，一个是“闭合”的，跟字母 C 和 O 也是非常契合。
